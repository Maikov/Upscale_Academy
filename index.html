A list of 4 tutorials I like for teaching HTML/CSS and Javascript.

  1. htmlbook.ru
    2. like:

    A very friendly tool while working, you can check quickly if you do't remember smth.

    dislike:

    Theoretical only material. It is not a full information in some cases.

  2. codecademy.com
    3. like:

    There is a lot of useful and relevant materials, very friendly material

    dislike:

    It is not suitable during the work, you need a lot of extra time to spend

  3. puzzleweb.ru
    4. like:

    A lot of relevant background information that can be used during the work. It helps with HTML/CSS JS together and it useful for beginners

    dislike:

    It can be used as a theory only

  4. learn.javascript.ru
    5. like:

    Good tutorial for beginners, there is a lot of practice. It can be used for both theory and practice.

    dislike:

    Short of code examples. It's difficult to use when working quickly.
